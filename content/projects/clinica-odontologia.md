A modern, minimalist, and clean landing page template for dental clinics. This project features a fully responsive design with optimized performance and easy configuration system.

## Key Features

- **Modern Design**: Minimalist and clean aesthetic perfect for dental clinics
- **Fully Responsive**: Mobile-first approach with complete responsive design
- **Optimized Performance**: Built with Next.js App Router for optimal performance
- **Image Integration**: Unsplash image integration for high-quality visuals
- **Easy Configuration**: Centralized configuration system for quick customization
- **Accessible**: Accessible components following best practices
- **SEO Optimized**: Built with SEO best practices in mind

## Technology Stack

- Framework: Next.js 14 with App Router
- Language: TypeScript
- Styling: Tailwind CSS
- Images: Next/Image with Unsplash integration
- Configuration: Centralized clinic configuration system

## Project Structure

The project includes components for Header, Hero, About, Specialties, Medical Team, Testimonials, Contact, Location, and Footer. All clinic-specific content is centralized in a configuration file for easy customization.

A comprehensive dental clinic landing page featuring appointment booking functionality, medical team showcase, specialties display, and integrated location services.

## Key Features

- **Appointment Booking**: Integrated appointment booking form for patient scheduling
- **Medical Team Showcase**: Display of medical professionals and their expertise
- **Specialties Section**: Showcase of dental specialties and services offered
- **Location Integration**: Clinic location with map integration
- **Testimonials**: Patient testimonials section
- **Contact Information**: Comprehensive contact section
- **Modern UI**: Built with shadcn/ui components for consistent styling

## Technology Stack

- Framework: Next.js with App Router
- Language: TypeScript
- Styling: Tailwind CSS v4
- UI Components: shadcn/ui (Button, Card, Input, Label, Textarea)
- PostCSS: For CSS processing

## Components

The project includes comprehensive components:
- Header with navigation
- Hero section
- About section
- Medical Team showcase
- Specialties display
- Appointment booking form
- Location with map
- Contact information

- Footer
